# QA API Testing Project

##  Technologies
- Python
- Pytest
- Requests

##  Test Scenarios
- Get users list
- Get single user
- User not found (404)
- Create user

##  Validations
- Status codes
- Response body
- JSON structure
- Headers validation

## Description
This project tests a REST API using Python and Pytest. It validates endpoints, responses and error handling, simulating real QA backend testing.

## How to run
pip install -r requirements.txt
pytest
